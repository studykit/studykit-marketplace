## Naming Conventions

- Most sprites use lowercase with hyphens: `docker-icon`, `google-cloud`
- Some use underscores: `apache_cloudstack`
- `-icon` suffix usually means a compact/square icon version
- Without suffix is typically the full logo/wordmark
