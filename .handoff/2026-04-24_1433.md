## Design decisions made during the rewrite

These shaped the output but were judgment calls not explicit in the ADR. Flag them if next session wants to reconsider.

1. **State derivation from workspace, not frontmatter.** `think-plan` has no `phase:` / `revision:` / `status:` field on `plan.md`. Mode (plan / implement / iterate) is computed at session start from per-task `status:` values. Tasks left in `implementing` from a crashed prior session are auto-reset to `pending` on resume (resume hygiene rule in `think-plan/SKILL.md`).
2. **Test failures as review items.** `test-runner` emits one `kind: finding` review item per failing test (with `source: test-runner`, `labels: [test-failure, cycle-<N>, tier:<integration|smoke>]`). There is no aggregated `test-report.md` artifact. `think-plan` then classifies each item as task/plan vs arch vs usecase and routes accordingly.
3. **Wiki pages carry `kind:` + `updated:` only.** No `revision`, `sources`, or `reflected_files` — cross-reference consistency is the drift detector's responsibility (script is still TBD; auto-bootstrap has a placeholder manual check in Step 8).
4. **`domain-updater` agent not invoked from `think-arch` anymore.** The skill edits `domain.md` directly with the wiki update protocol. The agent file is preserved but unused; it can be retired in a future cleanup if nothing else calls it.
5. **Auto-usecase source attribution goes in UC body, not frontmatter.** Each auto-written UC has a `## Source` section (`input | research — <systems> | code — <path> | implicit`). No `source:` frontmatter field on UCs — that keeps the ADR's minimal UC schema intact.
6. **`plan.md` Dependency Graph is a snapshot.** The authoritative source is per-task `depends_on:` frontmatter; the PlantUML block in `plan.md` is labeled as such. Regenerate via compass/dataview when tasks change.
7. **Bootstrap archiving moved to `a4/archive/`.** Old pattern was `a4/<slug>.bootstrap.r<N>.md`. New pattern on iteration: copy existing `a4/bootstrap.md` to `a4/archive/bootstrap-<YYYY-MM-DD-HHMM>.md` before writing the new one. The ADR's rejected-alternatives list already forbids `archived: true` frontmatter, and folder location is the flag.
